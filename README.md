# Deepsea-Compounds
深海深渊化合物及相关靶标的数据（受2021年海南省重大科技计划项目 任务-深海深渊微生物活性物质资源研究-ZDKJ2021028资助）

目标：收集已报道深海深渊化合物及相关靶标的数据。

方法：通过对ChemBL、ZINC 等小分子化合物库进行来源筛选，通过检索Uniport、TTD 等数据库查询检索遴选出有药物报道或临床前研究的靶标或者细胞组成药物靶标库，通过检索PubMed、Google Scholar进行数据来源补充和确认。

结果：目前共搜集获得了310 种深海深渊化合物（详见 表2 324种深海来源化合物信息.docx）和80 种相关靶标数据（详见 表1 深海深渊化合物的80种靶标信息）.
